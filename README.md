Here are some **basic Linux practice questions** that cover the most commonly used commands.

### File and Directory Commands

1. Create a directory named `linux_practice`.
2. Navigate into the `linux_practice` directory.
3. Create three empty files: `file1.txt`, `file2.txt`, and `notes.txt`.
4. List all files and directories.
5. Display the current working directory.
6. Create a subdirectory named `docs`.
7. Move `notes.txt` into the `docs` directory.
8. Rename `file1.txt` to `data.txt`.
9. Copy `data.txt` to `backup.txt`.
10. Delete `backup.txt`.

### Viewing File Contents

11. Add the text "Hello Linux" to `data.txt`.
12. Append the text "Learning Linux" to `data.txt`.
13. Display the contents of `data.txt`.
14. Show the first 5 lines of a file.
15. Show the last 5 lines of a file.

### Permissions

16. Check the permissions of `data.txt`.
17. Make `data.txt` executable.
18. Remove execute permission from `data.txt`.
19. Give read, write, and execute permission to the owner only.

### Searching

20. Find a file named `data.txt`.
21. Search for the word "Linux" inside `data.txt`.
22. Count how many times the word "Linux" appears in a file.

### File Operations

23. Create a file with 10 lines of text.
24. Count the number of lines, words, and characters in the file.
25. Sort the contents of a file alphabetically.
26. Remove duplicate lines from a file.
27. Compare two files.

### Compression

28. Create a compressed archive of the `docs` directory.
29. Extract the archive.

### User Information

30. Display the current logged-in user.
31. Display your user ID and group ID.
32. Show who is currently logged into the system.

### System Information

33. Display the Linux kernel version.
34. Display disk usage.
35. Display free memory.
36. Display CPU information.
37. Display system uptime.

### Process Management

38. List all running processes.
39. Find the PID of the `bash` process.
40. Kill a process by its PID.
41. Display processes in real time.

### Networking

42. Display your IP address.
43. Test internet connectivity using `ping`.
44. Display active network connections.

### Redirection and Pipes

45. Save the output of `ls` into a file.
46. Append the output of `date` to the same file.
47. Count the number of files in the current directory using a pipe.
48. Display only `.txt` files using `ls` and `grep`.

### Advanced Beginner

49. Create a file using `touch`.
50. Create a file using `echo`.
51. Create a file using `cat`.
52. Display command history.
53. Clear the terminal screen.
54. Create nested directories in a single command.
55. Remove a non-empty directory.
56. Create a symbolic link to `data.txt`.
57. Display the absolute path of the current directory.
58. Change to the previous directory using a single command.
59. Find the largest file in a directory.
60. Combine multiple commands using `&&`.

These 60 questions are excellent practice for beginners and cover about **80–90% of the Linux commands commonly asked in interviews and lab exercises**.
